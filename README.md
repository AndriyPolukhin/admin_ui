# Admin UI

A general purpose Admin UI created with:

1. [create-react-app](https://create-react-app.dev/)
2. [material ui](https://mui.com/material-ui)
3. [nivo charts](https://nivo.rocks/components)
4. [full calendar](https://fullcalendar.io/docs)
5. [formik](https://formik.org/docs/overview)
6. [yup validation](https://github.com/jquense/yup)
7. [react pro sidebar](https://github.com/azouaoui-med)
8. [google fonts](https://fonts.google.com/)
